üí° Why does larger kernel_size produce smaller output?

  • Kernel is a sliding window that moves across the input image
  • Each position the kernel covers produces exactly 1 output value
  • Larger kernel = fewer positions it can fit = smaller output
  • Stride controls how many pixels to move each step; larger stride = smaller output
  • Padding adds zeros around the edges, which increases output size
